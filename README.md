# Assessment App 
<br>
Features <br>
- Login with OTP <br>
- Different Forms <br>
- User can Save Form to DB so that even if they close and reopen the app the Data is present <br>
- User can Share the Form through different platforms <br>
- User can Save the form to local Storage <br>
